Gulp install:

1. install nodejs: https://nodejs.org/en/
2. install gulp: https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md
3. install ruby: http://rubyinstaller.org/
4. install sass: http://sass-lang.com/
5. заходишь в терминале в папку, где лежит gulpfile.js и пишешь npm install
6. установятся пакеты, и пропиши в этой же папке gulp
